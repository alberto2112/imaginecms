$_CURRENT

Type: array

Content:

  • LAYOUT:
  • SECTION: Array
    • NAME: String, code of page/section
    • LABEL: String, Label of page/section
    • APP: String, Name of associanted APP
    • ENABLED: Integer
      • 0 - page/section disabled
      • 1 - page/section enabled and visible
      • 2 - page/section enabled but hidden
    • APP: String [SECTION][APP]... but ... array is better (NAME, PATH, URL)

$_ERROR

Type: array

Content:

  • CODE: String, error code.
  • CONTENT: String, error description
